## Job Description Parser

A Python script that extracts structured information from job descriptions and exports the results into CSV format for analysis.

## Features

- Extracts company names using regex-based matching
- Identifies job roles and technical skills
- Processes multiple job descriptions in a single session
- Saves extracted data into CSV format
- Modular extraction functions for easier maintenance

## Project Structure

```text
main.py
job_info.csv
```

## Tech Stack

- Python
- Regex
- CSV
- File Handling
- Git & GitHub

## Running the App

- Clone the repository
- Run `python main.py`
- Paste job descriptions into the terminal
- Type `done` to finish
- View extracted data in `job_info.csv`
