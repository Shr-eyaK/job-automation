# Job Description Parser

A Python script that extracts company names, job roles, and technical skills from job descriptions and saves them to a CSV file. Supports job trend analysis.

## Technologies Used
Python, Regex, CSV, Data Analysis, Git

## Installation
1. Clone the repository  
2. Run `python main.py`  
3. Paste job descriptions and type 'done' when finished  
4. Check `job_info.csv` for output
